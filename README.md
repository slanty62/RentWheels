# 🚗 RentWheels

> Минималистичный веб-сервис для аренды автомобилей.  
Быстрый поиск, удобное бронирование и прозрачные цены.

![Версия](https://img.shields.io/badge/версия-1.0.0-blue)
![Лицензия](https://img.shields.io/badge/лицензия-MIT-green)
![Статус](https://img.shields.io/badge/статус-активен-brightgreen)

---

## ✨ Возможности

- 🔍 **Поиск** по марке, модели, году и цене  
- 📅 **Календарь** для выбора дат аренды  
- 💳 **Мгновенное бронирование** без звонков  
- 👤 **Личный кабинет** с историей поездок  
- 🛠️ **Админ‑панель** для управления автопарком  
- 📱 **Адаптивный дизайн** для любых устройств  

---

## 🚀 Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/slanty69/rentwheels.git

# Перейти в папку проекта
cd rentwheels

# Установить зависимости (frontend + backend)
npm install
cd client && npm install && cd ..

# Запустить в режиме разработки
npm run dev
