## ✨ Особенности

- **Поиск автомобилей** по множеству параметров (марка, модель, год, тип кузова, коробка передач, цена в день)
- **Интерактивный календарь** для выбора дат аренды с проверкой доступности
- **Мгновенное бронирование** без подтверждения менеджером (если авто свободно)
- **Личный кабинет** — история бронирований, отзывы, избранные авто
- **Панель администратора** — CRUD для автомобилей, пользователей, просмотр броней
- **Автоматический расчёт стоимости** с учётом количества дней и возможных скидок
- **Отзывы и рейтинг** автомобилей (средняя оценка от пользователей)
- **Фильтрация и пагинация** через query-параметры
- **JWT‑аутентификация** (access + refresh токены)
- **Поддержка изображений** (загрузка фото автомобилей через облачное хранилище S3 или локально)
- **Кэширование** часто запрашиваемых данных (Redis)
- **Асинхронность** (FastAPI + async SQLAlchemy)

## 🧰 Технологический стек

| Компонент          | Технологии |
|--------------------|-------------|
| **Язык**           | Python 3.11+ |
| **Web‑фреймворк**  | FastAPI |
| **ORM**            | SQLAlchemy 2.0 (асинхронный) |
| **База данных**    | PostgreSQL 15+ |
| **Кэш**            | Redis |
| **Миграции**       | Alembic |
| **Аутентификация** | python-jose (JWT), passlib (bcrypt) |
| **Валидация**      | Pydantic v2 |
| **Фоновые задачи** | Celery + Redis (отправка email, генерация отчетов) |
| **Тестирование**   | pytest, httpx (асинхронный клиент) |
| **Контейнеризация**| Docker, Docker Compose |
| **Документация API** | Swagger UI (встроена в FastAPI) |
